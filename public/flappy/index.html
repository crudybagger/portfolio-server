How to Use this Correctly ?

This Neuro Evolution version of the game will train itself.
You can control the speed of training with the sildier given below the game graphics frame(set to 20% by default).
Higher Speeds may cause Frames to Drop on some devices!!!
It only takes about 30 seconds of runtime (on 100% speed) to see Visible Changes.
After training set the speed to Lowest to see the magic.
